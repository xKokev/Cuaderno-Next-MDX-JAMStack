# Header o cabeceras H1 a H6
Para escrbir cabeceras en archivos markdown se realizan de la asiguiente manera dependiendo lo que tamaÃ±o titulo necesites
```md:cabeceras.mdx
# H1 Header
## H2 Header
### H3 Header
#### H4 Header
##### H5 Header
###### H6 Header
```
# Italics
```md:italics.mdx
*1 Prueba de en fuente italics*
Tambien funciona asi 
_2 Prueba de en fuente italics_
```
# Negritas o Bold

```md:bold.mdx
**Bold**
```
# Links - URL

```md:link.mdx
[Link](https://github.com/xKokev)
```

# Items

```md:items.mdx
* item 1

2. item 2
```

# Frases o quotes
```md:quote.mdx
> Quote text
```

# Inline code
```md:inlineCode.mdx
`Codigo en linea`
```
